1. Install python 3.8 or higher on your system.
2. Clone this repository. Repository link : https://github.com/ImranChowdhuryFahim/OpenCV.git
3. Inside the OpenCV folder you will find a requirements.txt file. Use “pip3 install -r requirements.txt” command to install required modules.
4. To use Android/IOS phone camera as webcam install Droidcam. 
   Droidcam windows client: https://www.dev47apps.com/droidcam/windows/
   Droidcam Linux client:
   https://www.dev47apps.com/droidcam/linux/

   Droidcam for Android devices: 
   https://play.google.com/store/apps/details?id=com.dev47apps.droidcam
   Droidcam for IOS devices: 
   https://apps.apple.com/us/app/droidcam-wireless-webcam/id1510258102

5. Run the completemouse.py file and use colored cap on your index and thumb finger.
